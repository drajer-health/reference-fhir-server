Observation

Observation APIs enable users to search and fetch vitals, smoking status & lab result data associated with a patient.

Observation.read

Http Method:

GET

Url Template:

[baseURL]/Observation/{ID}

Observation.search

Http Method:

GET

Url Template:

[baseURL]/Observation?patient={patient}&code={code}&category={category}&date={date}&status={status}

Conformance Parameter (Combination) Type URL
SHALL patient+code reference+token [baseURL]/Observation?patient={patient}&code={code}
SHALL patient+category reference+token [baseURL]/Observation?patient={patient}&category={category}
SHALL patient+category+date reference+token+date [baseURL]/Observation?patient={patient}&category={category}&date={date}
SHOULD patient+code+date reference+token+date [baseURL]/Observation?patient={patient}&code={code}&date={date}
SHOULD patient+category+status reference+token+token [baseURL]/Observation?patient={patient}&category={category}&status={status}

Supported API's: InteropX APIs are supporting following categories for Observation -

Category Description
social-history The status of a patient's tobacco use. This could be used to determine if the patient is a current or former tobacco user, what type of tobacco product is/was used, and the dates the patient started and, if a former user, stopped using tobacco.
vital-signs This will retrieve vital sign data from a patient's chart. Clinical observations measure the body's basic functions such as blood pressure, heart rate, respiratory rate, height, weight, body mass index, head circumference, pulse oximetry, temperature, and body surface area.
laboratory The results of observations generated by laboratories. Laboratory results are typically generated by laboratories providing analytic services in areas such as chemistry, hematology, serology, histology, cytology, anatomic pathology (including digital pathology), microbiology, and/or virology. These observations are based on analysis of specimens obtained from the patient and submitted to the laboratory.
