What is InteropX?

The InteropX platform instantly enables compliance with the CMS Interoperability and Patient Access Rule, the ONC 21st Century Cures Act, and with the U.S. Core Data for Interoperability (USCDI), providing certified APIs and apps for the Patient Access Portal with Provider Directory and Drug Formulary. InteropX is ready for the upcoming Provider Burden Reduction regulations, including Coverage Requirement Discovery (CRD), Documentation Templates and Rules (DTR), and Prior Authorization Support (PAS).
