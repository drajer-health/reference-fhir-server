Patient

Patient APIs enable user to fetch basic demographics and other administrative information about an individual patient.

Patient.read

Http Method:

GET

Url Template:

[baseURL]/Patients/{ID}

Patient.search

Http Method:

GET

Url Template:

[baseURL]/Patient?_id={id}&identifier={identifier}&name={name}& birthdate={birthdate}&family={family}&gender={gender}

Conformance Parameter (Combination) Type URL
SHALL _id token [baseURL]/Patient?_id={id}
SHALL identifier token [baseURL]/Patient?identifier={identifier}
SHALL name string [baseURL]/Patient?name={name}
SHALL birthdate+name date+string [baseURL]/Patient?name={name}&birthdate={birthdate}
SHALL gender+name token+string [baseURL]/Patient?name={name}&gander={gender}
SHOULD family+gender string+token [baseURL]/Patient?family={family}&gender={gender}
SHOULD birthdate+family date+string [baseURL]/Patient?family={family}&birthdate={birthdate}
