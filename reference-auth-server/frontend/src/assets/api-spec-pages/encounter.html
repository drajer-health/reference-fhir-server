Encounter

Encounter APIs enable users to search and fetch basic encounter information for an individual patient.

Encounter.read

Http Method:

GET

Url Template:

[baseURL]/Encounter/{ID}

Encounter.search

Http Method:

GET

Url Template:

[baseURL]/Encounter?&_id={id}&patient={patient}&status={status}&category={category}&date={date}&type={type}&period={date}

Conformance Parameter (Combination) Type URL
SHALL _id token [baseURL]/Encounter?_id={id}
SHALL patient reference [baseURL]/Encounter?patient={patient}
SHALL patient+date reference+date [baseURL]/Encounter?patient={patient}&category={category}
SHOULD identifier token [baseURL]/Encounter?id={identifier}
SHOULD patient+status reference+token [baseURL]/Encounter?patient={patient}&status={status}
SHOULD patient+class reference+token [baseURL]/Encounter?patient={patient}&class={class}
SHOULD patient+status reference+token [baseURL]/Encounter?patient={patient}&type={type}
