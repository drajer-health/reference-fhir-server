Organization

Organization APIs enable users for searching and fetching Organization associated with a patient or provider.

Organization.read

Http Method:

GET

Url Template:

[baseURL]/Organization/{ID}

Organization.search

Http Method:

GET

Url Template:

[baseURL]/Organization?name={name}&address={address}

SHALL name string [baseURL]/Organization?name={name}
SHALL address string [baseURL]/Organization?address={address}
