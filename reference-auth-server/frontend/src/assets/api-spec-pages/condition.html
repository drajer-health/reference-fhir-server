Condition

Condition APIs enable users to search and fetch a list of conditions associated with a patient.

Condition.read

Http Method:

GET

Url Template:

[baseURL]/Condition/{ID}

Condition.search

Http Method:

GET

Url Template:

[baseURL]/Condition?patient={patient}&code={code}&onset-date={onset-date}&clinical-status={clinical-status}&category={category}

Conformance Parameter (Combination) Type URL
SHALL patient reference [baseURL]/Condition?patient={patient}
SHOULD patient+code reference+token [baseURL]/Condition?patient={patient}&code={code}
SHOULD patient+onset-date reference+date [baseURL]/Condition?patient={patient}&onset-date={onset-date}
SHOULD patient+clinial-status reference+token [baseURL]/Condition?patient={patient}&clinical-status={clinical-status}
SHOULD patient+category reference+token [baseURL]/Condition?patient={patient}&category={category}

Supported API's: InteropX APIs are supporting following categories for Condition -
