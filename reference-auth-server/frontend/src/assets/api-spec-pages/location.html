Location

Location APIs enable users for searching and fetching a Location associated with a patient, provider or organization.

Location.read

Http Method:

GET

Url Template:

[baseURL]/Location/{ID}

Location.search

Http Method:

GET

Url Template:

[baseURL]/Location?name={name}&address={address}&address-city={address-city}&address-state={address-state}&address-postalcode={address-postalcode}

Conformance Parameter (Combination) Type URL
SHALL name string [baseURL]/Location?name={name}
SHALL address string [baseURL]/Location?address={address}
SHOULD address-city string [baseURL]/Location?address-city={address-city}
SHOULD address-state string [baseURL]/Location?address-state={address-state}
SHOULD address-postalcode string [baseURL]/Location?address-postalcode={address-postalcode}
