DiagnosticReport

DiagnosticReport APIs enable users to search and fetch laboratory results and notes associated with a patient.

DiagnosticReport.read

Http Method:

GET

Url Template:

[baseURL]/DiagnosticReport/{ID}

DiagnosticReport.search

Http Method:

GET

Url Template:

[baseURL]/DiagnosticReport?patient={patient}&category={category}&date={date}&code={code}&status={status}

SHALL patient reference [baseURL]/DiagnosticReport?patient={patient}
SHALL patient+category reference+token [baseURL]/DiagnosticReport?patient={patient}&category={category}
SHALL patient+category+date reference+token+date [baseURL]/DiagnosticReport?patient={patient}&category={category}&date={date}
SHALL patient+code reference+token [baseURL]/DiagnosticReport?patient={patient}&code={code}

SHOULD patient+status reference+token [baseURL]/DiagnosticReport?patient={patient}&status={status}
