Goal

Goal APIs enable users to search and fetch Goal information associated with a patient.

Goal.read

Http Method:

GET

Url Template:

[baseURL]/Goal/{ID}

Goal.search

Http Method:

GET

Url Template:

[baseURL]/Goal?patient={patient}&target-date={date}&lifecycle-status={lifecycle-status}

SHALL patient reference [baseURL]/Goal?patient={patient}
SHOULD patient+target-date reference+date [baseURL]/Goal?patient={patient}&target-date={target-date}
SHOULD patient+lifecycle-status reference+token [baseURL]/Goal?patient={patient}&status={lifecycle-status}
