Procedure

Procedure APIs enable users to search and fetch procedures associated with a patient.

Procedure.read

Http Method:

GET

Url Template:

[baseURL]/Procedure/{ID}

Procedure.search

Http Method:

GET

Url Template:

[baseURL]/Procedure?patient={patient}&date={date}&code={code}&status={status}

Conformance Parameter (Combination) Type URL
SHALL patient reference [baseURL]/Procedure?patient={patient}
SHALL patient+date reference+date [baseURL]/Procedure?patient={patient}&date={date}
SHOULD patient+code+date reference+token+date [baseURL]/Procedure?patient={patient}&code={code}&date={date}
SHOULD patient+status reference+token [baseURL]/Procedure?patient={patient}&status={status}
