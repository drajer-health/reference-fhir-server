DocumentReference

DocumentReference APIs enable users for searching and fetching patient documents including Clinical Notes.

DocumentReference.read

Http Method:

GET

Url Template:

[baseURL]/DocumentReference/{ID}

DocumentReference.search

Http Method:

GET

Url Template:

[baseURL]/DocumentReference?_id={id}&patient={patient}&status={status}&category={category}&date={date}&type={type}&period={date}

Conformance Parameter (Combination) Type URL
SHALL _id token [baseURL]/DocumentReference?_id={id}
SHALL patient reference [baseURL]/DocumentReference?patient={patient}
SHALL patient+category reference+token [baseURL]/DocumentReference?patient={patient}&category={category}
SHALL patient+category+date reference+token+date [baseURL]/DocumentReference?patient={patient}&category={category}&date={date}
SHALL patient+type reference+token [baseURL]/DocumentReference?patient={patient}&type={type}
SHOULD patient+type+period reference+token+date [baseURL]/DocumentReference?patient={patient}&type={type}&period={period}
SHOULD patient+status reference+token [baseURL]/DocumentReference?patient={patient}&status={status}
