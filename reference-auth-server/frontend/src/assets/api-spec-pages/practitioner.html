Practitioner

Practitioner APIs enable users to search and fetch basic demographics and other administrative information about an individual practitioner.

Practitioner.read

Http Method:

GET

Url Template:

[baseURL]/Practitioner/{ID}

Http Method:

GET

[baseURL]/Practitioner?name={name}&identifier={identifier}

SHALL name string [baseURL]/Practitioner?name={name}
SHALL identifier token [baseURL]/Practitioner?identifier={identifier}
